Data Exporter

The Data Exporter allows you to export Model Category data into CSV formats for BOQ and QA-QC purposes.

  • BOQ Export: Aggregates quantities by length, area, volume, or count. You can also breakdown items by level.
  • QA-QC Export: Extracts precise instance data alongside their respective parameters for validation workflows.

Quality Check / Import

Use the Quality Check tool to validate your parameter values and check compliance before exporting.

Do your Engineering and .CSV and import back to revit and BOOM.

Import Note: The CSV import process will only work for Shared Parameters, Project Parameters, and Instance Parameters. Type parameters and built-in system parameters cannot be imported.
